Mysql_skill_concat

Jun. 25, 2020

MySQL查询某一列列值拼接起来

需求

查询某一列并把列值拼接起来，拼接符用逗号

实现

借助于GROUP_CONCAT()函数的SQL语句：

SELECT
	  GROUP_CONCAT(concat("'",a.`fieldName`,"'")) AS fieldNameStr
FROM
	`tableName` a 
WHERE
	 a.`fieldState` = 1

注意事项

如果拼接的列值字符串长度超过了默认的1024，就会自动截断，只保留前面1024个字符，所以使用时需要注意这点。

如果确认超过了1024的默认长度，先查看系统设置的长度

SHOW VARIABLES LIKE 'group_concat_max_len';

如果group_concat_max_lenvalue 值 小于拼接字符的长度，则有两种方法：

  • 临时设置group_concat_max_len的最大长度值，修改MySQL的配置文件group_concat_max_len = 102400，设置后重启服务 > SET GLOBAL group_concat_max_len = 102400; > > SET SESSION group_concat_max_len = 102400;

  • 永久设置group_concat_max_len的value值，设置后重启服务

设置完成后，再执行查询语句就好了，不会截断语句了！
